Comandos SQL: 

USE (Nome do banco a ser utilizado) - Comando utilizado para abrir um Banco e poder executar comandos. 
LIKE (%nome do trecho a ser procurado%) - Nesse trecho o metodo like e um auxiliador de pesquisa, pois ele buscara onde contem a palavra digitada apos o metodo.
Se o % for antes da palavra, a palavra que esta procurando tem que estar no final, se for antes ela tem que estar no inicio. Se for nas duas pontas ele ira buscar onde houver a palavra.						
SHOW DATABASES - Seleciona os bancos ativos. 
; = Encerra o comando. 
CREATE SCHEMA ('nome do banco de dados') - Cria um banco de dados. 
DROP SCHEMA ('nome do banco de dados') - Deleta o banco de dados. 
CREATE DATABASE (Nome do banco de dados) - Cria um banco de dados. 
START TRANSACTION - Backup para poder voltar a alteracao.
ROOLBACK - Volta a alteracao, caso o programador tenha usado o START TRANSACTION. 

Explicando o comando abaixo:

SELECT * FROM (Nome da tabela) Where (Nome da coluna)

Select = Seleciona o trecho indicado
* = Tudo que contem na tabela 
From = indica de onde ir buscar os dados. 
Nome da tabela = Sera utilizado para identificar a qual tabela o codigo esta se referindo. 
Where = Indica de onde na tabela ele vai buscar o assunto.
Nome da coluna = Assunto no qual o select tem objetivo consultar.
